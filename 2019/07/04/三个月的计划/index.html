三个月的计划

发表于 2019-07-04

给自己定了一个三个月的计划

提升英语能力

提升英语这个说法其实是个很虚无缥缈的东西。如何提升？提升一点点也叫提升，你的目标是什么？

首先是目标（长期目标）：

  • 以托福100分+的水平为目标（是指实力，而不一定要去考托福）
  • 能够熟练的与歪果仁进行日常交流、面试交流

接下来是提升的方法：

  • 单词学习（一周三次，每次1h）
  • 看老友记并尝试shadowing（一周两次每次约1h）
  • 每天阅读economist上的一篇文章，并尝试完全理解文章意思

锻炼身体

自己的身体素质在经过了长年累月的躺尸之后已经下降到了一种非常差劲的地步。需要进行康复训练。

  • 俯卧撑两组（一组10个，一周4次）
  • 仰卧起坐两组（一组20个，一周4次）
  • 乒乓球或者游泳（一周1次，每次2h）
  • 跑步（一周2次，每次0.5h）

阅读

今天摸鱼看leader的书《程序员修炼之道——从小工到专家》时看到了leader在13年写的笔记，leader给自己定了目标一个月看两本书（技术、非技术各一本）。其实我很早就有这个打算，但是自己的自律能力实在是太差了，今天才开始执行。

  • 技术书（一本 x 3 = 3本）
  • 非技术书（一本 x 3 = 3本）
  • 一个月写上两篇的读书笔记

喜欢的事，算法竞赛题

之前荒废了太久了，该重新拿起来了

  • 每周3题CF的2000难度的题

控制时间

  • 少看微博
  • 少看知乎

后记：阅读、锻炼身体和算法竞赛题的目标对于现在我来说完成起来还是有难度的。三个月后来进行一次该计划的总结，不知道自己能完成多少呢？
