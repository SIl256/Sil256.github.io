题意

给定二维平面上 $1 \le N \le 1000$ 个整点 $(X_i, Y_i), -10^9\le X_i,Y_i\le 10^9$ ,有上下左右$(U,D,L,R)$四种操作，求一个序列$D\{d_0,d_1,\cdots,d_n\}$以及N个长度为n的操作序列$\{op_{i,0}, op_{i, 1}, \cdots, op_{i,n}\}$ ，使得对于第 $i$ 个操作序列，从$(0, 0)$出发每次按照操作序列里的操作顺序和序列 $D$ 里的距离走，能走到点 $(X_i, Y_i)$

分析

用 {$2^x: 0 \le x \le 30$} 可以构造出 $(M, 0), (-M, 0), (0, M), (0, -M), M = 2^{30 + 1}-1$，围起来的四边形内任意一个满足 $|x + y| \equiv 1 (mod 2)$ 的点，$|x + y| \equiv 0(mod2)$的点同理，再额外加个1到集合中即可

102
C: 水题
D:
题意

给定整数 $2 \le L \le 10^6$，构造一个有向图满足下面四个条件

  • 点最多 $20$，下标从 $1$ 到 $20$
  • 边最多 $60$，每条边的权值范围为 $[0,10^6]$
  • 每条边都从编号小的点指向编号大的点
  • 刚好有 $L$ 条从点 $1$ 到点 $N$ 的路径，且这些路径的长度分别为 $[0, L-1]$

分析

点下标 $0 \cdots m$，点 $i$ 向 $i+1$ 连两条边权值分别 $0$ 和 $2^i$，现在可以构造出 $[0, 2^m)$ 的所有路径
离最终想要的路径数之差为 $n - 2^m$，然后用二进制拆位的思想，从 $0 \cdots m-1$ 中的一些点向 $m$ 连边，下标为 $i$ 的点连的边能够多出 $2^i$ 条不同的路径

E: 组合数学
题意
有 $2 \le N \le 2000$ 个一模一样的骰子，每个骰子有$ 1 \le K \le 2000 $ 面，每面上的数字分别是 $[1,K]$，对于$ i = 2,3,\cdots,2K$, 求满足如下条件的方案数：

  • 任意两个骰子加起来的和不等于 $i$

分析
对于一种情况来说
假设 $t$ 为奇数，可以将 $n$ 个数字分成两类
一类是 $a + b = t(a和b都在1..k的范围内)$ 的数字
除了上述 $(a, b)$ 以外的数字称为第二类数字
第一类数字中的数字在答案中只会出现 $(a, b)$ 二者之一，而对于第二类数字是没有限制的
假设第一类 $pair(a, b)$ 数量有 $p$ 种，在答案中出现了 $q$ 种，这样的方案数是 $C(p, q) \times 2^q$
问题可以转换成将 $n$ 个球，分到两类盒子里
第一类盒子里的盒子(有 $q$ 个盒子)必须分到至少一个球
第二类盒子里的盒子(有$k - 2 \times p$)可以分到任意个球，令 $m = k - 2 \times p$
隔板法计算，方案数就是：$C(n + m - 1, q + m - 1)$
最后的答案就是：$C(p, q) \times 2^q \times C(n + m - 1, q + m - 1)$
当 $t$ 为偶数的情况第一类数字会出现 $a + a = t$ 这样的数，这样的数只会出现1或0个，分类讨论转换成 $t​$ 为奇数的情况

D:
题意
给定长度为 $1 \le N \le 10^5$ 的序列 $A$，对于所有的子段而言的中位数构成新序列 $B$，求B的中位数。
中位数为第 $(\lfloor Len/2 \rfloor + 1)$ 个数

分析
假设最终序列的长度为 $M$，中位数是 $x$，则 $\ge x$ 的数有$\lceil M \div 2 \rceil$个，并且 $x$ 是满足上述条件最大的数
基于上述结论，可以通过对最终序列的中位数 $x$ 进行二分，将原问题转化为判断多少个 $(l, r)$ 满足 $mid(l, r) \ge x$
新转化的问题就可以通过将原数组中的元素 $ \ge x$ 的设为$1$，$ \le x$设为 $-1$，$S$ 表示新数组前缀和，求新数组中$ (l, r) $满足$S_r-S_l \ge 0$的数量

题意
给定 $2N,(1\le N \le 2000)$ 个球排成一排，其中 $N$ 个黑球，$N$ 个白球，每个球的下标从$1, \cdots, N$, 每次可以 交换两个相邻 的球，问使得对于任意一种颜色的球，都满足按照下标递增顺序时最少进行多少次操作。

  2. current node have left child and the most right child of the left child is NULL
  3. current node have left child and the most right child of the left child is current node

dp

  2. the minimal number of one end is ‘)’

温暖的事物不会消失

发表于 2019-07-21

“也许对人类而言，这只是一起稍微有点水花的案件，可是对我而言，我的夏天，结束了啊。”

2019.07.18已经过去3天，但是在这几天夜里，睡前的我总是会想到7.18的那个恶性的事件。
我总想着，在几年前的那个夏天，我第一次接触到《冰菓》，被《冰菓》的画风、剧情、配乐惊艳，那个时候的我的心情。
我总想着，工作之后，要去圣地巡礼，要去拜访一下能做出这么温柔的动画的人们他们工作的地方、他们生活的地方。
我不是很深的宅，看的动画不多，重复看过好几遍的动画更少。在我为数不多刷过3-4遍的动画里，只有京阿尼的《冰菓》和《日常》。
制作出陪伴了我一个夏天动画的地方，就这样随着一把火，成了废墟的模样。罪犯仅仅是自己认为莫须有的“抄袭”，做出了这种事情，杂碎，不配为人。
难以想象为什么会有人对能够做出如此温柔的动画片的人抱有强烈的恶意，明明大家都是很温柔的人啊。
718事件之前，我对京阿尼的了解仅仅存在于京阿尼的动画很对我的胃口。
718之后，我才深入了解了京阿尼。了解京阿尼的前身，了解京阿尼对动画的追求，了解京阿尼对待员工的态度。
很抱歉，在这种时候我才能去了解。

关于培训遇到的些许事

首先谈谈在培训的某一天进行的某一个游戏时遇到的事情：
先说一下整件事情的大致过程，这是一个七巧板游戏，总共有6个组，每个组还必须出一个人组成第7个组。在这7个组之中，只有第7组的成员可以四处的走动，而其他组的成员统统不能离开位置。
我就成为了第7组的成员。游戏的目标大概就是需要完成各自小组的目标和拼出7个组的不同七巧板图形，第7小组的目标是领导其他小组拼出正方形。
这时候让我疑惑的事情就出现了，第7组的成员其实只有第7组的利益而没有原先小组的利益，因为这时我们只属于第7组，而不属于原先的小组。尽管如此，我们之中还是有人一直在倾斜资源向他们原先所在的小组。
另一件让人疑惑的事情是在一开始强调了不能使用手机，但还是有很多的人通过手机将7个组的七巧板图形分享出去了。
第三件事是在拼大正方形的时候，拼不出来竟然拿出了手机查拼法。

我并没有很强的集体荣誉感，我觉得用集体来绑架个人是很傻逼的行为，同时我也是很遵守规则的人。后两件事情，让我很失望，为什么大家就没办法好好的遵守规则呢？玩游戏的前提就是大家都遵守规则，一旦有人不遵守规则，那么很快这个游戏就会有很多人一起不遵守规则，最后导致严重的后果甚至毁掉这个游戏，例如APEX。这就是所谓的破窗户效应。这第三件事我觉得是相当的丢人了，一旦遇到了一些个人利益相关的抉择，有些人就会抛弃诚信，我是相当的讨厌这样子的人。也许是我太上纲上线了，只是一个游戏而已。

第二件事是关于我的交际能力：
在培训期间被拉去了参与某个活动的策划，后来又去参加了班里的舞蹈节目，被之前策划组中的一位小姐姐认出来加了微信好友，然后在聊了没两句之后我直接“跟女孩子聊天太难了，再见！”终止聊天，我真是。我的交际能力在大学的四年里有着显著的下降，一方面是长期混迹于代码之中，另一方面是自己过于内向。我不知道该跟新认识的人聊些什么，导致我有点孤僻。就像cc所说的那种“我知道你知道，跟我不知道你知道”，在我不知道他们知道我的一些想法时，很多的事情我是不想讲出来，这样子就会导致我的话题很少，话题少的同时又会导致我对他们的理解不够，最终死锁。以后要尝试跟新认识的人多聊聊天呀。

后记

本篇就是关于培训几天的所见所闻所想，并没有什么意义。
